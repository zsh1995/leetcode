# 数一
## 问题描述
给定一个`int`型变量，统计其二进制表示中`1`的个数。
## 解决方法
### 直接法
`int`型变量为 `32 bit`，最多直接向右移位 32次，统计出现`1`的次数。
### 数学技巧
考虑如下：<br/>
```
 x    =          1011 0000
x - 1 =          1010 1111
x & ( x - 1 ) =  1010 0000
```
可以看到 `x & (x - 1)` 可以消掉最后一个 `1`。利用这个技巧可以快速跳过多个0。
#### 由此技巧的衍生
如何快速判断一个数是不是 2 的幂？一个快速的判别法即是 `return x & (x - 1) == 0`
